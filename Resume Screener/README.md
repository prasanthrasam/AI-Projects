# AI Resume Screener using Langflow

## Project Overview

The AI Resume Screener is a Langflow-based application that automates the resume screening process. It analyzes candidate resumes against a job description (JD) using a Large Language Model (LLM) and provides an objective evaluation based on predefined hiring criteria.

The application reduces manual screening effort, improves consistency, and helps recruiters quickly identify suitable candidates.

---

## Objectives

- Automate resume screening.
- Compare resumes against a Job Description.
- Calculate candidate-job match percentage.
- Identify technical and functional skill gaps.
- Recommend whether the candidate should proceed to the next interview stage.
- Generate a structured evaluation report.
